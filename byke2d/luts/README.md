# luts

Lookup tables copied and converted from bevy. To skip the ktx2 dependency (as of
writing, there seems to not be one for golang), the lookup tables are converted
to raw blobs.

## Tony McMapface

```
vkFormat: VK_FORMAT_E5B9G9R9_UFLOAT_PACK32
typeSize: 4
pixelWidth: 48
pixelHeight: 48
pixelDepth: 48
```
